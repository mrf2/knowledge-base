# Astro Sequence (Mental Model)

 1. `npm create astro@latest` $\rightarrow$ sets the compiler
 2. We write `.astro` files $\rightarrow$ input to compiler
 3. `npm run dev` $\rightarrow$ live preview
 4. `npm run build` $\rightarrow$ final static HTML

We **never** run `.astro` files directly
They are **source code**, not runtime code.
