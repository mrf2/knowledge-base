# SNI - Server Name Indication 

### What is a hostname?
A hostname is the name of a device that connects to a network. In the context of the Intenet, a domain, or the name of a website, is type of hostname. Both are separate from the IP address associated with the domain name.

### What is a virtual hostname?
A virtual hostname is a hostname that doen't have its own IP address and is hosted on a server along with other hostnames. It is "virtual" in that it doesn't have a dedicated physical server, just as virtual reality exists only digitally, not in the physical world.
