# Laravel and Intertia Project

 1. `git clone <repo-url>`
 2. `cd <project-folder>`
 3. `composer update`
 4. `cp .env.example .env`
 5. `php artisan key:generate`
 6. `php artisan migrate`
 7. `npm install`
 8. `npm run dev` [in one terminal - maybe not mandatory]
 9. `php artisan serve`

---
# Core Commands (Quick Recall)
```bash
php artisan serve             # run backend
php artisan migrate           # apply schema
php artisan config:clear      # reset config
npm run dev                   # frontend (Inertia)
```
